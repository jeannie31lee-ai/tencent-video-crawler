# 腾讯视频片库爬虫

爬取腾讯视频电影和电视剧片库数据，经九层过滤后输出结构化 Excel 文件，包含独播/非独播标识、付费类型、男女频等维度。

## 数据概览（V10）

|------|----------|----------|----------|------|----------|
| 电视剧 | 4,517 | 855 | 3,662 | 2,005 | 54.8% |
| 电影 | 16,155 | 1,265 | 14,890 | 909 | 6.1% |

## 项目结构

```
├── scripts/
│   ├── crawl.py                 # 全量爬虫（年份×排序全交叉 + VID时长检测 + checkpoint）
│   ├── scan_vertical_drama.py   # 页面扫描（竖屏短剧/微短剧/无资源检测）
│   ├── filter_and_gen_excel.py  # 九层过滤 + 独播修正 + Excel 生成
│   └── fix_exclusive.py         # 独播修正规则定义（用户确认名单）

└── output/
    └── 腾讯视频片库数据.xlsx      # 最终 Excel 输出
```

## 使用方式

### 1. 安装依赖

```bash
pip install requests openpyxl
```

### 2. 全量爬取

```bash
python scripts/crawl.py
```

爬虫通过腾讯视频 `getMVLPage` API 全维度交叉爬取：

- 电视剧：11 个年份段 x 3 种排序 + 撞上限年份按 9 个地区拆分
- 电影：20 个年份段 x 3 种排序
- 自动检测独播标记（`latest_mark_label` 中 `tag_2`）
- 通过 `ipay` 参数交叉比对确定付费类型（会员/免费/付费）
- 支持 checkpoint 断点续跑，中间数据保存到 `/tmp/`

### 3. 页面扫描（可选，用于增强过滤）

```bash
python scripts/scan_vertical_drama.py
```

使用 20 线程并行扫描所有影剧的页面 HTML，检测竖屏短剧、微短剧、无资源等标记。扫描 18,686 页约 10 分钟。

### 4. 过滤 + 生成 Excel

```bash
python scripts/filter_and_gen_excel.py
```

加载爬取数据和扫描结果，依次应用九层过滤规则，执行独播修正，生成 Excel（全部详情 + 汇总统计两个 Sheet）。

## 九层过滤规则

| 规则 | 说明 | 电视剧 | 电影 |
|------|------|--------|------|
| Rule 1 | 预告片/外站/待播（positive_trailer=2） | 123 | 1,149 |
| Rule 2 | 横屏短剧（content_from=8375890，仅电视剧） | 710 | - |
| Rule 3 | 已下架（页面 404/内容移除） | - | 9 |
| Rule 4 | 宣传片/非正式内容（标题黑名单） | 1 | - |
| Rule 5 | 竖屏短剧/微短剧（页面扫描标记） | 17 | - |
| Rule 6 | 幕后纪录片（main_genre=纪录片 + 关键词，仅电影） | - | 99 |
| Rule 7 | 无资源（页面扫描 + 手动补充） | - | 4 |
| Rule 8 | 附属特辑/番外篇（标题正则匹配） | 2 | 2 |
| Rule 9 | 创意短片/短剧/微电影（标题关键词匹配） | 2 | 2 |

## 独播修正规则

- **用户确认独播电视剧**（约 40 部）：陈情令、庆余年、狂飙、三体、长相思等
- **用户确认非独播**：香蜜沉沉烬如霜、三生三世十里桃花、姐妹情缘等
- **电影年份截断**：2016 年前标记为"独播"的电影统一改为"非独播"
- **独播定义**：国内网络播放平台独播，不含海外平台（Netflix/WeTV）及电视台（央视/卫视）

## 技术要点

- **API 接口**：`pbaccess.video.qq.com` 的 `getMVLPage` 接口
- **页面扫描**：SSR 页面 `__NEXT_DATA__` JSON 提取 `positive_trailer` 字段
- **独播检测**：`latest_mark_label` JSON 中 `position 2` 的 `id=15, text=独播`
- **多维度爬取**：年份段 x 排序 x 地区 交叉爬取后 CID 合并去重
- **多线程扫描**：ThreadPoolExecutor 20 workers 并行页面扫描
- **正则过滤**：附属特辑/番外篇、短片/微电影使用正则匹配
