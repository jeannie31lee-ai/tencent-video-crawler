# 腾讯视频片库爬虫

爬取腾讯视频电影和电视剧片库数据，输出结构化 Excel 文件，包含独播/非独播标识、付费类型、男女频等维度。

## 数据概览

| 类型 | 总数 | 独播 | 独播占比 |
|------|------|------|----------|
| 电视剧 | 3,426 | 1,908 | 55.7% |
| 电影 | 7,479 | 499 | 6.7% |
| **总计** | **10,905** | **2,407** | **22.1%** |

## 项目结构

```
├── scripts/
│   ├── crawl.py            # 多维度爬虫（按年份+排序方式交叉爬取，合并去重）
│   ├── fix_exclusive.py    # 独播标识修正（用户反馈 + 规则修正）
│   └── gen_excel.py        # Excel 生成（全部详情 + 汇总统计两个 Sheet）
├── data/
│   └── tencent_video.json  # 最终数据（10,905 条）
└── output/
    └── 腾讯视频片库数据.xlsx # 最终 Excel 输出
```

## 使用方式

### 1. 爬取数据

```bash
pip install requests openpyxl
python scripts/crawl.py
```

爬虫通过腾讯视频 API 多维度爬取：
- 电视剧：17 个年份段 × 2 种排序方式，合并去重
- 电影：全量排序 + 年份补漏
- 自动检测独播标记（API `latest_mark_label` 中的 `tag_2`）
- 过滤单集 ≤20 分钟的短剧/短片
- 输出 JSON 到 `/tmp/tencent_video_v5.json`

### 2. 修正独播标识

```bash
python scripts/fix_exclusive.py
```

修正规则：
- **电影**：2016 年前的"独播"标签不可信（老电影在 1905 电影网/优酷等平台普遍有播出），统一改为非独播
- **电视剧**：保持 API 原始标记，叠加用户手动修正
- 独播定义：**国内网络播放平台独播**，不含海外平台（Netflix/WeTV）及电视台（央视/卫视）

### 3. 生成 Excel

```bash
python scripts/gen_excel.py
```

输出两个 Sheet：
- **全部详情**：平台、剧名、类型、是否独播、付费类型、年份、演员、地区、集数、题材、男女频
- **汇总统计**：按类型/付费/男女频/地区/年份多维度统计

## 技术要点

- **API 接口**：`pbaccess.video.qq.com` 的 `getMVLPage` 接口
- **独播检测**：`latest_mark_label` JSON 中 `position 2` 的 `id=15, text=独播`
- **短剧过滤**：通过 `h5vv6.video.qq.com/getinfo` 获取首集时长，≤20 分钟排除
- **付费类型**：通过 `ipay` 筛选参数交叉比对确定（会员/免费/付费）
- **多维度爬取**：避免单一排序方式的 1000 条上限，通过年份段 + 多排序合并去重
